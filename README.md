# 🚀 Universe Saver

**Universe Saver** is a fast-paced 2D survival shooter game where the player fights against waves of enemies to protect the universe.

Built using Unity, the game focuses on responsive controls, lightweight performance, and simple but engaging gameplay.


---

## 📥 Download

👉 **Google Play Store:**  
[Download Universe Saver](https://play.google.com/store/apps/details?id=com.Saquib.Universe&pcampaignid=web_share)

## 🛠️ Tech Stack

- **Game Engine:** Unity  
- **Language:** C#  
- **Platform:** Android  

---

## 📂 Project Purpose

This project was built to:

- Learn core game development fundamentals  
- Implement shooting mechanics and enemy systems  
- Understand game loops and player interaction  
- Gain experience publishing on Google Play Store

## 🎮 Gameplay

- Control your character in a hostile environment  
- Shoot and eliminate incoming enemies  
- Survive as long as possible  
- Difficulty increases over time

- 🎯 Progressive enemy difficulty
